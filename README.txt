# Proyecto: Rescate Inteligente

Este proyecto permite comparar distintos algoritmos de búsqueda para encontrar rutas de rescate desde hospitales hacia zonas de emergencia en una ciudad representada como grafo.

## Estructura

- `main.py`: Script principal que ejecuta las búsquedas y visualiza los resultados.
- `grafos/ciudad_ejemplo.json`: Grafo de ejemplo con posiciones y pesos.
- `algoritmos/`: Implementaciones de BFS, UCS, A* (Greedy a completar).
- `utils/`: Carga del grafo y visualización de caminos.

## Requisitos

Instalar las dependencias con:

```bash
pip install -r requirements.txt
```

## Ejecución

```bash
python main.py
```
